感谢您使用本主题！

Motion是一款简洁的暗色调博客主题，炫彩背景支持多种半透明阴影效果，配合最新0.9版Typecho模版结构，完美兼容全线浏览器：IE6+, FireFox 2+, Opera 9+和Safari 3+。

本主题版权归原作者所有, 由羽中(Jasper)移植至Typecho博客平台，遵循GPL开源共享协议.

你可以参考本主题的Wordpress演示并在这里下载和了解到最新的Typecho版本和使用说明。
